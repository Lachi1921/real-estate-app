# Real Estate App

[![Next.js](https://img.shields.io/badge/Next.js-%3E%3D13.0-blue)]()

A Next.js 13+ TypeScript web application for real estate listings. The project provides a framework for building a property listing platform with modern frontend features, optimized fonts, and TypeScript support. The app uses Next.js routing and server components to manage pages and content efficiently.

---

## Overview

* View **property listings**
* Navigate between pages using **Next.js routing**
* Use optimized **web fonts**
* Develop and test the application with **TypeScript** support

---

## Features

* **Next.js 13+ Framework**: App uses the latest Next.js routing and server components.
* **TypeScript**: Strongly typed frontend code for better developer experience.
* **Linting with ESLint**: Ensures clean and consistent code style.

---

## Tech Stack

* **Frontend:** Next.js 13+ (React-based)
* **Language:** TypeScript
* **Linting:** ESLint


## Installation

### 1. Clone the repository

```bash
git clone https://github.com/Lachi1921/real-estate-app.git
cd real-estate-app
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
